## Cloud Observability

### AWS
- **Amazon CloudWatch:** Native monitoring and observability service.
- **AWS X-Ray:** Distributed tracing.
- **AWS Distro for OpenTelemetry:** Secure, production-ready OTel distribution.
- [AWS Observability Best Practices](https://aws-observability.github.io/observability-best-practices/)

### Google Cloud (GCP)
- **Google Cloud Operations Suite:** (formerly Stackdriver) Monitoring, Logging, Trace, and Profiler.
- [Google Cloud Observability Architecture](https://cloud.google.com/stackdriver/docs/solutions/architecture)

### Microsoft Azure
- **Azure Monitor:** Full stack monitoring service.
- **Azure Application Insights:** APM for web developers.
- [Azure Observability Documentation](https://learn.microsoft.com/en-us/azure/azure-monitor/overview)
